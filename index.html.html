Detalhamento Técnico

Como Cada
Sistema Funciona.

A engenharia Pondus opera em camadas — cada subsistema é otimizado individualmente e integrado com precisão cirúrgica ao conjunto.

Sistema de Pesagem Gravimétrico
As células de carga strain-gauge de duplo cisalhamento medem a deformação elástica do elemento sensor. O conversor A/D de 24-bit digitaliza o sinal a 1.600 Hz, aplicando filtros digitais anti-aliasing e compensação de não-linearidade por tabela de lookup para atingir ±0,01% de exatidão sobre a faixa nominal.
Controle de Dosagem de Alta Velocidade
O algoritmo de previsão de corte antecipa o ponto exato de fechamento do gate com base na velocidade de queda do material, eliminando o efeito de material em queda livre. Isso permite operar a 600 sacos/hora mantendo desvio padrão inferior a 2g em ensacamento de 50kg.
Interface HMI e Conectividade Cloud
Painel touch 10" com IP65, resolução 1280×800. Dados de pesagem são enviados via MQTT ao PONDUS Cloud, onde dashboards em tempo real exibem OEE, tendências de desvio e alertas preditivos. API REST disponível para integração com qualquer ERP/MES.
Manutenção Preditiva com IA
Modelos de machine learning treinados com dados de 500+ equipamentos detectam padrões de degradação em rolamentos, válvulas e células de carga com 30 dias de antecedência, reduzindo paradas não programadas em até 78%.
